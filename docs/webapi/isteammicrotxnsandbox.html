ISteamMicroTxnSandbox Interface

Exposes the ISteamMicroTxn interface with a dummy database for development and testing purposes.

This interface is identical to the regular ISteamMicroTxn interface, but no actual transactions will occur.

See Microtransactions (In-Game Purchases) for more information.

For more info on how to use the Steamworks Web API please see the Web API Overview.
