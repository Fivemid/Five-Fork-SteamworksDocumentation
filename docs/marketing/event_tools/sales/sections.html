Redigerare för reasidor: Sektionstyper

Läs dokumentationen om redigeraren för reasidor för mer information om hur man konfigurerar en reasida.

Detta dokument beskriver de lika sektionstyperna som du kan använda på din reasida för att visa upp olika slags innehåll.

Standardsektioner

  • Appar och buntar – Denna är för applikationer, paket eller buntar. De kan inordnas i rutnät eller karuseller, eller så kan de visas en och en om du så föredrar.
  • Flikar – Om du har ett event med fler än 50–100 spel så föredrar du kanske att dela upp innehållet i olika flikar. Du kan använda flikar för att organisera din sida efter genre, kategori, tema eller något annat sätt som känns logiskt för spelen som är inkluderade i ditt event.
  • Textsektion – Här kan du ange och formatera text med BBCode-verktyg. Detta kan vara effektfullt och flexibelt om du vill visa tabeller, YouTube-videor, bannerbilder eller textblock.
  • Sändningsspelare – Här spelas sändningar upp från de spel som finns på din reasida. Vilken sändning som inleder avgörs slumpmässigt. Obs: Om du använder sändningsfliken ovan till att ange specifika konton får de kontona prioritet i sändningsspelaren och visas då först i ordningen.
  • Eventprogram – Detta kan användas till att visa schemalagda event under en vald tidsperiod för spel på din sida. Om det finns ett aktivt event av typen ”livestream” prioriteras det spelets sändning i sändningsspelaren.
  • Länkar – Lägg till länkar till andra delar av Steam-butiken. Du kan lägga till en bild till varje länk. Dessa kan inordnas i rutnät eller karuseller, eller som en enskild bannerbild.
  • Event – Detta är event som läggs ut via Steams verktyg för event och tillkännagivanden.
  • Kurator- eller skaparsida – Här visas en sektion med vald webbsida för kurator/utvecklare/utgivare, tillsammans med en följ-knapp och eventuella länkade konton för social media.

Experimentella sektionstyper

Dessa sektionstyper måste först aktiveras av Valve.
  • Webbläsare för rabatterade produkter – Användbart vid stora event då den aktiverar en fliksektion för att automatiskt visa spel efter ”bästsäljare”, ”nya släpp”, ”mest önskade”, ”populära” osv.

Sektionsalternativ

Varje sektion har några alternativ att välja bland.
  • Sektionstitel – Du kan välja från en lista med titlar som redan är översatta och klara, eller så kan du ange en egen titel. Se till att du sparar din egen titel separat genom att klicka på "Spara titel" bredvid den.
  • Undertitel – Det går också att ha en undertitel i de flesta sektionerna om du vill förklara något ytterligare i någon sektion.
  • Bakgrundsinställningar – Du kan anpassa bakgrundsfärgen för varje sektion och även lägga till en bild, om du så önskar. Detta kan göra det lättare att särskilja de olika sektionerna.
  • Flikinställningar – Om du använder dig av flikar på din sida så kan du specificera vilka flikar varje sektion ska synas under. Som standard visas en sektion under alla flikar om du inte specificerar under vilka de ska visas.

Tillägg av en sektion

För att lägga till en sektion på din sida ska du klicka på den blå Lägg till sektion-knappen långt ned under reafliken. När du väl lagt till en sektion kan du använda rullgardinsmenyn som visas nedan för att specificera vilken typ av innehåll som den sektionen ska innehålla.

Sändningsspelare

Här spelas sändningar upp från de spel som finns på din reasida. Vilken sändning som inleder avgörs slumpmässigt. Obs: Om du använder sändningsfliken ovan till att ange specifika konton får de kontona prioritet i sändningsspelaren och visas då först i ordningen.

Eventprogram

Detta kan användas till att visa schemalagda event under en vald tidsperiod för spel på din sida. Eventen som visas är ordnade efter starttid. Om flera event börjar samma tid blir ordningen slumpmässigt vald. Om det finns ett aktivt event av typen ”livestream” prioriteras det spelets sändning i sändningsspelaren som ”I fokus”. Sändningar av spel som är inkluderade i rean som inte uttryckligen står i programmet får standardprioritet.

I redigeraren för reasidor finns kontroller för att justera tidsintervallet för programmet, så att det kan skilja sig från den underliggande rean. Specifik eventtyp kan också konfigureras här.

Det går inte att begränsa antalet gånger ett spel kan synas i programmet. Du kan behöva koordinera/kommunicera en begränsning på hur många event olika spel ska skapa för att det ska vara lika för alla deltagare.

Om du vill göra tillägg (av till exempel ett konferensinlägg) till de schemalagda eventen som hör till underliggande festivaler istället för de angivna spelen, behöver du skapa en ”serie” via programverktyget. Serier tillåter att man lägger till publicerade event direkt genom de offentliga webbadresserna. Detta innebär att de schemalagda eventen måste publiceras innan du kan lägga till dem i programmet. Helst bör du skapa dessa event från samma gemenskapsgrupp eller skaparsida som är värd för din reasida. Varje inlägg på programmet kräver ett annat event, vilket innebär att repriser av en livestream måste ha ett eget offentligt event.

Flikar

Om du har ett event med fler än 50–100 spel så föredrar du kanske att dela upp innehållet i olika flikar. Du kan använda flikar för att organisera dina sida efter genre, kategori, tema eller något annat sätt som känns logiskt för spelen som är inkluderade i ditt event.

Flikar som filter

När flikar är aktiverade kan du använda varje flik för att filtrera innehåll genom att tillämpa en filterinställning på själva flikkonfigurationen.
Du kan ange enskilda taggar som filter, eller så kan du använda vissa funktioner. Här följer några exempel:
Filternamn Funktion
[Feature] HasDemo Filtrerar och visar endast basspel som har en giltig och synlig demo tillgänglig.
[Feature] Discounted Filtrerar och visar endast spel som har en aktiv rabatt.
[Feature] VR Filtrerar och visar endast spel med VR-stöd.
[Feature] Free Filtrerar och visar endast spel som är gratis och gratis att spel.
[Feature] Coming Soon Filtrerar och visar endast spel som inte är släppta.
