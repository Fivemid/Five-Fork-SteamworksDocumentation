Sales Landing Page Editor (Beta) - Section Types

This document will discuss the various types of sections you can use on your sale page to display different content.

Adding a Section
To add a section to your page, click the blue 'add section' button near the bottom of the Sale tab. Once you add a section, you can use the drop-down highlighted below to specify what kind of content should be in that section.

Items

These are apps, packages, or bundles. They can be organized into grids or carousels or can display a single item if you wish.

Events

These are events posted via the Steam Events & Announcements tool

Carousel of three items



Section of two items

Links

Add links to other parts of the Steam Store. For each link, you can provide an image. These can be organized into grids or carousels or just a single banner image.

Broadcast Player

This will play the broadcasts from any of the games listed within your sale page. It will randomize which broadcast to start on. Note: If you use the broadcast tab mentioned above to specify accounts, those accounts will take priority within the broadcast player and will appear first in the listing.

Event Description

This will display the same text that you entered on the 'Description' tab.

Text Section

This will let you enter and format text with bbcode tools. This can be pretty powerful and flexible if you need to display tables, YouTube Videos, banner images, or just blocks of text.

Curator or Creator Home Section

This will display a section with the selected Steam Curator or developer/publisher homepage, along with a 'follow' button and any linked social media accounts.

Event Schedule

This can be used to display events scheduled during the selected times by games listed on your page. If there is an active event with type 'livestream', that game's broadcast will take priority in the broadcast player.

Tabs

(This section type must first be enabled by Valve)
If you have an event with more than 50-100 games, you probably want to split the content up into different tabs. You can use tabs to organize your page by genre, category, theme, or any other way that makes sense for the games included in your event.

Sale Item Browser

(This section type must first be enabled by Valve)
Also useful for large events with many games, the Sale Item Browser will enable a tabbed section for automatically listing games by 'top seller', 'new release', 'most wishlisted', 'trending', and others
